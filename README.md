# Weight-Converter-GUI-Python
A simple Python Weight Converter GUI application built using Tkinter. Enter weight in kilograms and instantly convert it into grams, pounds, and ounces. Useful for beginners learning Python GUI development.

This is a simple Weight Converter application built using Python Tkinter. 
Enter weight in kilograms and convert it into:
- Grams
- Pounds
- Ounces
